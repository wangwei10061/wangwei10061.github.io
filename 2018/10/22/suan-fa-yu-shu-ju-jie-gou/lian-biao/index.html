Reverse Linked List 

Reverse a singly linked list.

Example:

Input: 1->2->3->4->5->NULL
Output: 5->4->3->2->1->NULL

题解：
有递归和迭代两种解法，其中迭代解法又分为两种，就地逆置法和头插法，区别在于头插法使用头结点，二者时间复杂度均为O(n)，空间复杂度均为O(1)

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode reverseList1(ListNode head) {
        // 使用就地逆置法
        ListNode nextNode = head;
        ListNode newHead = null;

        while (head !=null) {
            nextNode = nextNode.next;
            head.next = newHead;
            newHead = head;
            head = nextNode;
        }

        return newHead;
    }

    public ListNode reverseList2(ListNode head) {
        // 使用头插法
        ListNode newHead = new ListNode(0);
        ListNode nextNode = head;

        while (head != null) {
            nextNode = nextNode.next;
            head.next = newHead.next;
            newHead.next = head;
            head = nextNode;
        }

        return newHead.next;
    }

    public ListNode reverseList3(ListNode head) {
        // 递归解法
        ListNode newHead = null;

        // 到达最后一个节点时，递归返回
        if (head == null || head.next == null) {
            return head;
        } else {
            // 递归当前节点的下一节点
            newHead = reverseList(head.next);
            // 把当前节点的下一个节点的next指针指向当前节点
            head.next.next = head;
            // 当前节点的next指针置空
            head.next = null;
            return newHead;
        }
    }
}
LeetCode 92.

Reverse Linked List II

Reverse a linked list from position m to n. Do it in one-pass.

Note: 1 ≤ m ≤ n ≤ length of list.

Example:

Input: 1->2->3->4->5->NULL, m = 2, n = 4
Output: 1->4->3->2->5->NULL

题解：
跟前面那一题比较像，主要是处理逆置段前面部分，逆置段与逆置段后面部分，抓住连接处的结点关系即可解决问题

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode reverseBetween(ListNode head, int m, int n) {
        int reverseLen = n - m + 1;
        ListNode result = head;
        ListNode preReverse = null;

        // 处理逆置段前驱
        while (--m > 0) {
            preReverse = head;
            head = head.next;
        }

        // 最开始逆置的元素将是逆置后的最后一个元素
        ListNode reverseLast = head;

        // 逆置逆置段
        ListNode newHead = null;
        ListNode nextNode = head;
        for (int i = 0; i < reverseLen; i++) {
            nextNode = nextNode.next;
            head.next = newHead;
            newHead = head;
            head = nextNode;
        }

        // 连接逆置段与逆置段后面部分
        reverseLast.next = head;

        // 连接逆置段与逆置段前面部分
        if (preReverse != null) {
            preReverse.next = newHead;
        } else {
            result = newHead;
        }

        return result;

    }
}
LeetCode 21.

Merge Two Sorted Lists

Merge two sorted linked lists and return it as a new list. The new list should be made by splicing together the nodes of the first two lists.

Example:

Input: 1->2->4, 1->3->4
Output: 1->1->2->3->4->4

题解：
此题比较简单，依次比较两个链表的当前元素，取小的插入当前链表，最后把有剩余的链表链过来即可

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
        ListNode result = new ListNode(0);
        ListNode nextNode = result;

        while (l1 != null && l2 != null) {
            if (l1.val < l2.val) {
                nextNode.next = l1;
                l1 = l1.next;
            } else {
                nextNode.next = l2;
                l2 = l2.next;
            }
            nextNode = nextNode.next;
        }

        if (l1 != null) {
            nextNode.next = l1;
        } else {
            nextNode.next = l2;
        }

        return result.next;
    }
}
LeetCode 160.

Intersection of Two Linked Lists

Write a program to find the node at which the intersection of two singly linked lists begins.


For example, the following two linked lists:

A:          a1 → a2
                   ↘
                     c1 → c2 → c3
                   ↗            
B:     b1 → b2 → b3
begin to intersect at node c1.


Notes:

If the two linked lists have no intersection at all, return null.
The linked lists must retain their original structure after the function returns.
You may assume there are no cycles anywhere in the entire linked structure.
Your code should preferably run in O(n) time and use only O(1) memory.

题解：
法1：
利用set的元素不可重复性，先将A链表的所有元素均存入集合，然后遍历B链表，找出第一个在集合中出现的元素即可(不满足O(1)空间的需求)
法2：
如果不使用现有的数据结构，可以先求出两个链表的长度，然后将长的链表移动两链表长度之差，然后同步移动两个链表，遇到第一个相同的结点为止（粗略的实现在LeetCode会超时）

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
import java.util.Set;
import java.util.HashSet;
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        // 用set集合的特性
        Set<ListNode> set = new HashSet<ListNode>();
        while (headA != null) {
            set.add(headA);
            headA = headA.next;
        }

        while (headB != null) {
            if (set.contains(headB)) {
                return headB;
            }
            headB = headB.next;
        }

        return null;
    }

    public ListNode getIntersectionNode1(ListNode headA, ListNode headB) {
        // 取A长度
        ListNode nextA = headA;
        int lenA = 0;
        while (nextA != null) {
            nextA = nextA.next;
            lenA++;
        }

        // 取B长度
        ListNode nextB = headB;
        int lenB = 0;
        while (nextB != null) {
            nextB = nextB.next;
            lenB++;
        }

        // 长链表先行差值步
        int sub = 0;
        if (lenA > lenB) {
            sub = lenA - lenB;
            while (sub > 0) {
                headA = headA.next;
                sub--;
            }
        } else {
            sub = lenB - lenA;
            while (sub > 0) {
                headB = headB.next;
                sub--;
            }
        }

        // 同步往前走找交点
        while (headA != null) {
            if (headA == headB) {
                return headA;
            }
            headA = headA.next;
            headB = headB.next;
        }

        return null;
    }
}
LeetCode 142.

Linked List Cycle II

Given a linked list, return the node where the cycle begins. If there is no cycle, return null.

Note: Do not modify the linked list.

题解：
法1：
使用快慢指针赛跑算法，快慢指针一开始都指向head，然后快指针以步长2向前移动，慢指针以步长1向前移动，两指针相遇可以判断有环，相遇之后让快指针从head以步长1出发，慢指针从相遇的位置继续前进，再次相遇的点，即为环的起点。
原理：设从head到环的起点的长度为a，环的起点到两指针第一次相遇的位置的长度为b，两指针第一次相遇的位置沿前进方向到环起点的长度为c，由于： a + 2b + c = 2 (a + b)，所以：a = c
法2：
使用set求解，遍历链表，将链表元素一次存入set，第一次遍历到已经出现在set中的元素即为环的起点

/**
 * Definition for singly-linked list.
 * class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
public class Solution {
    public ListNode detectCycle(ListNode head) {
        // 快慢指针法
        ListNode fast = head;
        ListNode slow = head;

        while (fast != null && slow != null) {
            // 快指针以2的速度向前移动，慢指针以1的速度向前移动，如果有环，两个指针会在环内相遇
            slow = slow.next;
            if (fast.next == null) {
                return null;
            }
            fast = fast.next.next;

            if (slow == fast) {
                // 两指针相遇后，把快指针重新移动head，以和慢指针相同的速度向前移动
                fast = head;
                while(fast != slow) {
                    fast = fast.next;
                    slow = slow.next;
                }
                return fast;
            }
        }

        return null;
    }
}
LeetCode 86.

Partition List

Given a linked list and a value x, partition it such that all nodes less than x come before nodes greater than or equal to x.

You should preserve the original relative order of the nodes in each of the two partitions.

Example:

Input: head = 1->4->3->2->5->2, x = 3
Output: 1->2->2->4->3->5

题解：
采用两个带头结点的新链表，一个存储比x小的结点，一个存储大于等于x的结点，然后连接即可

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode partition(ListNode head, int x) {
        // 采用两个头结点，一个存储比x大的元素，一个存储比x小的元素，遍历原来链表，按大小分别将每一个元素插入两个新的链表
        ListNode newSmallerHead = new ListNode(0);
        ListNode newBiggerHead = new ListNode(0);

        ListNode newSmallerNextNode = newSmallerHead;
        ListNode newBiggerNextNode = newBiggerHead;
        while (head != null) {
            if (head.val < x) {
                System.out.println("小" + head.val + " " + x);
                newSmallerNextNode.next = head;
                newSmallerNextNode = newSmallerNextNode.next;
            } else {
                System.out.println(head.val + " " + x);
                newBiggerNextNode.next = head;
                newBiggerNextNode = newBiggerNextNode.next;
            }
            head = head.next;
        }

        // 因为每遍历到一个结点是把包括该节点在内的整个后续链链接到新的链表，所以操作完之后需要删除后续附带的结点
        newSmallerNextNode.next = null;
        newBiggerNextNode.next = null;

        // 连接
        newSmallerNextNode.next = newBiggerHead.next;

        // 返回结果
        return newSmallerHead.next;
    }
}
LeetCode 138.

Copy List with Random Pointer

A linked list is given such that each node contains an additional random pointer which could point to any node in the list or null.

Return a deep copy of the list.

题解：
难点在于处理随机指针，不能直接复制，只能复制相对关系
使用两个map，map1存储原结点与节点在原链表中的相对位置（索引值）的映射关系
，map2存储节点在新链表中的位置与新链表结点的关系。那么，对于random
域，根据random域指向的结点在原链表中的位置，利用该位置在新链表中找到对应的结点，并链接过去，就完成了链表的深度拷贝。

/**
 * Definition for singly-linked list with a random pointer.
 * class RandomListNode {
 *     int label;
 *     RandomListNode next, random;
 *     RandomListNode(int x) { this.label = x; }
 * };
 */
import java.util.HashMap;
import java.util.Map;

public class Solution {
    public RandomListNode copyRandomList(RandomListNode head) {
        if (head == null) {
            return null;
        }

        RandomListNode nextNode = head;
        Map<RandomListNode, Integer> originalNodeIndexMap = new HashMap<RandomListNode, Integer>();

        RandomListNode newHead = new RandomListNode(0);
        RandomListNode newNextNode = newHead;

        Map<Integer, RandomListNode> newIndexNodeMap = new HashMap<Integer, RandomListNode>();

        int i = 0;
        while (nextNode != null) {
            // 创建新链表
            newNextNode.next = new RandomListNode(nextNode.label);

            // 源node加入map
            originalNodeIndexMap.put(nextNode, i);
            // 新node加入map
            newIndexNodeMap.put(i, newNextNode.next);
            nextNode = nextNode.next;
            newNextNode = newNextNode.next;
            i++;

        }

        newHead = newHead.next;

        // 重新遍历新链表，为新链表设置random指针域
        newNextNode = newHead;
        nextNode = head;
        while (nextNode != null && newNextNode != null) {
            if (nextNode.random != null) {
                Integer index = originalNodeIndexMap.get(nextNode.random);
                newNextNode.random = newIndexNodeMap.get(index);
            }
            nextNode = nextNode.next;
            newNextNode = newNextNode.next;
        }

        return newHead;

    }
}
