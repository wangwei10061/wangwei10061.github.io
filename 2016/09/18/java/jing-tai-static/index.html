语法：静态--static
JAVA
JAVA
发布日期:   2016-09-18
阅读次数:  

Java中有一个特殊的关键字，叫static关键字，一个变量或方法是否被static修饰会导致它的性质和应用范围有很大的不同。本文主要讨论Java中静态与非静态的区别。

老毕的视频教程中讲到：

static是一个修饰符，用以修饰类成员（成员变量，成员函数）

简明扼要的一句话，完美地诠释了static，静态与非静态的区别仅仅在于静态的属性或方法属于类，非静态的属性或方法属于某个具体的对象。

概要

  • 静态的特点
  • 实例变量（成员变量）与类变量（静态的成员变量）的区别
  • 静态何时使用
  • 静态使用注意事项
  • 使用静态的利弊
  • 运用实例

静态的特点

  1. 随着类的加载而加载，随着类的消失而消失
  2. 优先于对象而存在
  3. 可以被类名直接调用
  4. 被所有对象所共享

实例变量（成员变量）与类变量（静态的成员变量）的区别

  1. 存放位置：
    实例变量：存放于堆内存中
    类变量：存放于方法区中
  2. 生命周期：
    实例变量：随着对象的建立而存在，随着对象的消失而消失
    类变量：随着类的加载而存在，随着类的消失而消失

静态何时使用

  • 静态属性：当所有的对象共享同一属性时，将该属性用static修饰
  • 静态方法：当方法中不涉及访问具体对象的属性时用static修饰

静态使用注意事项

  1. 静态方法只能访问静态成员
  2. 静态方法中不能定义this，super等关键字

使用静态的利弊

  • 利： 对对象的共有数据局进行单独的存储，节省空间；可以直接用类名调用，方便
  • 弊： 访问出现局限性；生命周期长

运用实例
一般工具类多会封装一些静态的方法

//定义数组工具类ArrayTool
class ArrayTool {
 private ArrayTool(){}//将构造函数私有化，防止外界调用该类创建无意义的对象
 //找寻数组元素中最大值
 public static int getMax(int arr[]) {
  int max=0;
  for(int i=1;i<arr.length;i++) {
   if(arr[i]>arr[max])
    max=i;
  }
  return arr[max];
 }
 //找寻数组中最小的元素
 public static int getMin(int arr[]) {
  int min=0;
  for(int i=1;i<arr.length;i++) {
   if(arr[i]<arr[min])
    min=i;
  }
  return arr[min];
 }
 //排序
 public static void selectSort(int arr[]) {
  for(int i=0;i<arr.length-1;i++)
   for(int j=i;j<arr.length;j++) {
    if(arr[i]>arr[j]) {
     int temp;
     temp=arr[i];
     arr[i]=arr[j];
     arr[j]=temp;
    }
     //swap(arr[i],arr[j]);
   }
 }
 //将数组输出
 private static void swap(int a,int b) {
  int temp;
  temp=a;
  a = b;
  b=temp;
 }
 public static void showArray(int arr[]) {
  for(int i=0;i<arr.length;i++)
   System.out.print(arr[i]+" ");
  System.out.println();
 }
}



class ArrayTest {
 public static void main(String[] args) {
  int arr1[]={1,2,3,6,4,32};
  int max=ArrayTool.getMax(arr1);
  System.out.println("max="+max);

  ArrayTool.showArray(arr1);
  ArrayTool.selectSort(arr1);
  ArrayTool.showArray(arr1);
 }
}
